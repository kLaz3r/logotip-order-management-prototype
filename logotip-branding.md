# LogoTeep Web App Branding Specification

## Brand Overview

The visual identity is built around a modern print-and-graphics studio
aesthetic: - Bold geometric typography - Strong editorial layout -
Minimal white space with high contrast - Playful creative signal through
the stacked geometric icon - Professional production feel combined with
approachable design

## Brand Personality

Keywords: - Creative - Precise - Modern - Reliable - Design-focused -
Print craftsmanship

## Typography Style

Recommended direction: - Geometric sans-serif - Large bold display
headlines - Clean lowercase branding style - Rounded, approachable
letterforms

Suggested fonts: - Inter - Poppins - Montserrat - Avenir-style
alternatives

## Logo Usage Rules

-   Maintain generous whitespace around logo
-   Use purple as the dominant brand anchor
-   Avoid placing logo on visually busy backgrounds
-   Preserve icon proportions
-   Keep the geometric mark separate from cluttered UI elements

## UI Design Direction

### Layout

-   Minimal layouts
-   Large typography
-   Clear hierarchy
-   Strong alignment
-   Spacious sections

### Components

Buttons: - Rounded corners - Purple primary buttons - Orange/yellow
hover states

Cards: - White backgrounds - Soft shadows - Purple or teal accents

Backgrounds: - Primarily white - Occasional warm cream or light neutral
sections

## Illustration Style

Use: - Geometric shapes - Flat vector styling - Simple color blocking -
Print-inspired visual language

Avoid: - Heavy gradients - Photorealistic imagery - Excessive decoration
